==================

:ref:`frozenset`) unterscheiden sich im Wesentlichen durch folgende
Eigenschaften:

|               | <Unveränderlich>`     |               |               |               |

| Liste         | ✅                    | ✅            | ✅            | ✅            |
+---------------+-----------------------+---------------+---------------+---------------+
| Tuple         | ❌                    | ✅            | ✅            | ✅            |
+---------------+-----------------------+---------------+---------------+---------------+
| Set           | ✅                    | ❌            | ❌            | ❌            |
+---------------+-----------------------+---------------+---------------+---------------+
| Frozenset     | ❌                    | ❌            | ❌            | ❌            |
+---------------+-----------------------+---------------+---------------+---------------+
